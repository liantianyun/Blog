# 夜深了

“天运弟弟，现在有后线肠癌临床研究了，细胞治疗，在中肿，阿姨近期情况如何？”

妈妈去世前一两个月前，曾尝试加入临床试验，是这位小姐姐负责帮我们寻找实验组，联系医院的，然而临床试验的入组，对于患者的身体条件是有要求的，妈妈彼时的肺炎持续无法消退，医生评估不具备入组的条件，于是这个希望便也就破灭了。今天，突然这个小姐姐给我发了这么一条消息，她兴许是还不知道妈妈已经走了吧。我跟她说：“妈妈已经去世很久了，没有加入临床试验后，一两个月后去就去世了”。她说：“不好意思，项目来晚了”。

我不知道该对这句话作何感想，除了勾起我的伤心外，有何其它意义呢。

前段时间读书的时候，读到这么一段话：

> 笔者从事肿瘤治疗近40年，近年来颇有感触：随着肿瘤治疗的进步，如今欲逃脱癌症恶魔的纠缠已非难事。近40年来，癌症生存者日渐增多。以上海为例，患癌而活着者，从20世纪80年代初的区区两三万人，飙升到现在的36.8万人（据上海权威部门2018年4月16日报道），美国则接近1500万人；上海癌症患者5年相对生存率已达53%，而发达国家的5年生存率早已高达70%～80%。以我所在的肿瘤治疗专业机构（上海民生中医门诊部）为例，中西医结合手段治疗癌症，5年相对生存率达到83.1%（而且涉及数万例大样本），包括原本被认定是癌中之王、活不过两三个月的肝癌。今年毕业的赵博士，因为博士论文需要，统计分析了本机构资料完整的413例肝癌患者，结论是肝癌患者平均生存期110.5个月（9.2年），中位生存期75个月；曾做过手术切除肝癌并加中医药善后的，平均生存期（128.02±12.76）个月（10.68年），中位生存期92个月；1年、3年、5年生存率分别为95.80%、89.00%和85.40%。即使没法手术切除的晚期患者，中医药再加综合治疗（包括靶向药物）的99例患者，1年、3年、5年生存率分别为92.90%、76.50%和72.70%。这说明什么问题，说明今天癌症已经有好办法对付；好好治疗，死于癌症者会越来越少。诚如本书作者布来得森所言：“每个人都可能认识一些癌症康复者。”其实，每个人周边都有很多癌症康复者。你想想，全美国癌症生存者高达1500万，每20个美国人中（全美3亿人，不管男女老少）就有一位癌症康复者。上海虽尚未达到美国康复水平，也是每50人中有一位癌症康复者。因此，可以自信地说，生了癌，合理治疗，康复已不再是什么难事。人们大可不必再为癌症恐惧了！起码在发达城市，合理治疗，并配合有友好的支持性氛围条件，生了癌（包括像难治性的肝癌）还是能够好好活着的。

多么乐观的一段话呀，多么令人振奋的数据:`即使没法手术切除的晚期患者，中医药再加综合治疗（包括靶向药物）的99例患者，1年、3年、5年生存率分别为92.90%、76.50%和72.70%。`,早上妈妈患病初期，我就曾查到过多个资料表达过类似的言论，它们都说，只要积极治疗，是极有可能长期带癌生存的，可是如果真是这样的话，为什么妈妈就这么不幸运嗯？如果晚期肿瘤的五年生存率到已经高达了70%以上，妈妈为什么非要是那百分之三十呢？

连外婆都曾不知道是在安慰自己还是在安慰我的说：“这就是她的命”。

我去问chatGpt，为什么现在医生要这样宣扬，妈妈九个月多来的治疗经历在我眼前历历在目，在我的视角里，跟媒体宣扬的根本不一样，妈妈的所接受的化疗，客观上只是起到了延长生命时间的作用，或许，甚至没有，我有时不免会想，兴许当时不接受治疗，只是积极的去改变生活方式，或许妈妈拥有的时间会更加长吧？当化疗无法再起到任何作用的时候，妈妈在短短的一两个月内，就在我眼前我眼睁睁的看着她一天天变得虚弱，这时，作为一个普通人所能找到的所有治疗方式，已经无一能再对她起到任何的作用了，这时候，我深深的感觉到现代医学的局限和无力，它能治疗什么呢？它无所作为，让我一天天的看着妈妈的生命之花在我眼前凋谢，我不知是否有人能体会，那对于我而言是一种怎样的心情。

chatGpt说：“医生之所以这样宣扬是为了给那些患癌的人传递一个积极的信号，让他们建立起信心，积极的去接受治疗对抗癌症”。是呀，予人信心有什么错呢？这世界上如此多不幸的人患上了癌症，这些人在接触到这样积极的言论的时候，总比接触到“无望等死”之类的消极言论要来的强吧。

所以我也没有资格埋怨什么吧，我只能一再的去不甘，去问：“为什么不幸的人非要是她？”

外婆也许比我更加难接受吧？妈妈到死之前，都没能敢向外婆交代自己的病情，我无法想象，当已经八十几岁的外婆收到关于女儿的消息，便是她的死讯时，是以怎样的心情接受这一切的？

妈妈的死对我影响很大，我知道，我的生命不受自己所控制，兴许任何人都是如此，那些健康的人儿，转瞬之间健康状况便可一日千里，生命就是这样，充满了无奈，你只能去接受，有些事不可能会释怀。

奶奶也快八十岁了。

**好好珍惜你眼前的人吧，穷尽所有的能力去爱她。**

**不要让离去来教你成长，那是世界上最愚蠢的事情。**
